# 버그헌터(BugHunter) — AI 디버깅 교육 플랫폼 기획서 v1

## 1. 서비스 개요

**한줄 요약:** AI가 의도적으로 버그가 있는 코드를 생성하고, 학생이 버그를 찾아 고치는 과정에서 디버깅 사고력을 훈련하는 교육 플랫폼

**3줄 설명:**
- AI 시대에 코드 작성은 AI가 대신하지만, 버그를 찾고 고치는 능력은 여전히 개발자의 핵심 역량이다.
- 버그헌터는 학생 수준에 맞는 버그 코드를 AI가 자동 생성하고, 답을 주지 않는 단계별 힌트 시스템으로 디버깅 사고 과정 자체를 훈련시킨다.
- 강사는 수업 주제만 입력하면 디버깅 실습 문제가 자동 생성되고, 대시보드에서 학생들의 디버깅 역량을 한눈에 파악할 수 있다.

## 2. 문제 정의

### 타겟이 겪는 구체적 문제

**수강생의 문제:**
> "부트캠프에서 코드 짜는 건 배웠는데, 에러 나면 멘붕. 실무에서 디버깅을 못 해서 혼남"
> "브레이크포인트도 못 잡는 신입 개발자를 봤다"

- 모든 코딩 교육이 "코드 짜는 법"에만 집중하고, "코드 고치는 법"은 가르치지 않는다
- 실무 개발자의 하루 60~70%는 코드 읽기/디버깅에 소모된다 (대규모 코드베이스에서는 80~90%)
- AI가 코드를 더 많이 생성하는 시대에, 코드 읽기/디버깅 능력이 오히려 더 중요해지고 있다

**강사의 문제:**
> "강사 1명이 30명의 디버깅 과정을 일일이 봐줄 수 없다"

- 디버깅은 사고 과정이 핵심인데, 결과물만 봐서는 학생이 어디서 막히는지 알 수 없다
- 디버깅 실습 문제를 매번 수동으로 만드는 건 시간이 오래 걸린다

**교육기관의 문제:**
> "수료생이 실무에서 디버깅을 못 한다는 피드백이 채용 기업에서 온다"

- 취업률은 교육기관의 핵심 KPI인데, 수료생의 실무 적응력이 떨어지면 기업 평판에 악영향

## 3. 타겟 사용자

| 구분 | 메인 타겟 | 서브 타겟 |
|------|-----------|-----------|
| 역할 | 국비지원/부트캠프 코딩 강사 | 수강생 (비전공자 포함) |
| 연령 | 30~50대 | 20~30대 |
| 행동 특성 | 수업 준비에 시간 부족, 30명 개별 피드백 불가 | 에러 나면 바로 ChatGPT에 복붙, 디버깅 경험 전무 |
| 핵심 니즈 | 디버깅 실습 문제를 빠르게 만들고, 학생 현황을 파악하고 싶다 | 에러가 두렵지 않게 되고 싶다 |

## 4. 핵심 기능 (3개)

### 기능 1: AI 버그 코드 자동 생성
- 강사가 수업 주제(예: "반복문")를 입력하면, AI가 해당 주제의 버그가 포함된 코드를 3단계 난이도로 자동 생성
- 🟢 초급: 문법/타이포 오류 (예: 인덱스 범위 오류)
- 🟡 중급: 로직 오류 (예: 무한루프, off-by-one)
- 🔴 고급: 설계 오류 (예: 엣지 케이스 미처리, 비효율 알고리즘)
- 버그는 정확히 1개만 포함, 나머지 코드는 정상 작동
- 강사가 생성된 문제를 미리 확인하고 수정/승인 가능

### 기능 2: 단계별 힌트 시스템 (답을 주지 않는 AI)
- 학생이 막혔을 때, AI가 직접 답을 알려주지 않고 사고를 유도하는 힌트를 제공
- **Level 1 힌트 — 방향 제시:** "에러가 반복문 안에 있어"
- **Level 2 힌트 — 범위 좁히기:** "3번째 줄의 반복 조건을 확인해봐"
- **Level 3 힌트 — 개념 설명:** "이건 off-by-one 에러야. <=와 <의 차이를 생각해봐"
- **최종 (포기 시):** 정답 코드 + 왜 이것이 버그인지 설명 + 유사 연습 문제 제공
- 핵심: 학생이 직접 생각하고 찾아내도록 유도하는 구조

### 기능 3: 강사 대시보드 + 학생 성장 리포트
- **강사 화면:** 반 전체 학생의 디버깅 현황을 한눈에
  - "우리 반은 반복문 버그는 80%가 잡는데, 조건문 엣지케이스는 30%만 잡음"
  - 학생별 힌트 사용 횟수, 풀이 시간, 성공률
  - "이번 주 집중 지도 대상" 자동 추천
- **학생 화면:** 내 디버깅 성장 기록
  - "2주 전에는 문법 오류만 잡았는데, 이제 로직 오류도 잡아!"
  - 디버깅 스킬 레벨 시각화

## 5. 사용자 플로우

### 👨‍🏫 강사 플로우
```
1. 회원가입/로그인
2. 반 생성 ("Python 기초반") → 학생 초대코드 발급
3. 과제 출제: 주제 입력 ("반복문") → AI가 3단계 버그 문제 자동 생성 (10초)
4. 생성된 문제 미리보기 → 필요시 수정 → 출제 확정
5. 대시보드에서 학생별 풀이 현황 실시간 확인
6. 수업 후: "우리 반 디버깅 리포트" 확인 → 다음 수업 계획에 반영
```

### 👩‍💻 학생 플로우
```
1. 초대코드로 가입 → 반 입장
2. 오늘의 디버깅 과제 확인 → 난이도 선택 (또는 강사가 지정)
3. 웹 코드 에디터에서 버그 있는 코드 확인
4. 코드 읽기 → 버그 위치 파악 시도
5. 막히면 "힌트 요청" → Level 1 힌트 수령
6. 여전히 막히면 Level 2, Level 3 힌트 순차 수령
7. 버그 수정 → "제출" → 코드 실행으로 정답 확인
8. 성공 시: "버그 잡았어! 🎉" + 다음 문제 추천
9. 실패 시: 정답 해설 + 유사 연습 문제 제공
10. 내 성장 리포트 확인: 디버깅 스킬 레벨, 약한 영역 등
```

## 6. 차별화 포인트

### "코딩 퀴즈(백준/프로그래머스)와 뭐가 다른가?"

| 구분 | 백준/프로그래머스 | 버그헌터 |
|------|-------------------|----------|
| 목표 | 정답 코드를 처음부터 작성 | 기존 코드에서 버그를 찾아 수정 |
| 핵심 역량 | 코드 쓰기 (writing) | 코드 읽기 + 고치기 (reading + fixing) |
| 막혔을 때 | 정답/풀이 공개 | AI가 답 안 주고 단계별 힌트로 사고 유도 |
| 학습 추적 | 맞았다/틀렸다 | 어떤 유형의 버그를 못 잡는지 분석 |
| 교육 연동 | 독립 플랫폼 | 강사 수업 보조 도구 (주제 연동) |
| AI 역할 | 없거나 부수적 | 문제 생성 + 힌트 + 분석의 핵심 |

### "ChatGPT에 코드 붙여넣기와 뭐가 다른가?"

| 구분 | ChatGPT | 버그헌터 |
|------|---------|----------|
| 답을 주는가? | 바로 정답 코드를 줌 | 절대 안 줌. 힌트만 줌 |
| 학습 이력 | 매번 리셋 | 학생별 약점/성장 누적 추적 |
| 수업 연동 | 없음 | 강사가 주제 지정, 대시보드 확인 |
| 교육 목적 | 범용 도구 | 디버깅 사고력 훈련 전용 |

### 시대적 차별화
- "AI가 코드를 짜주는 시대, 개발자의 진짜 실력은 디버깅이다"
- Copilot/Cursor가 코드 작성을 대체할수록, 디버깅·코드 리딩 역량의 가치는 올라간다
- 기존 교육은 이 변화를 반영하지 못하고 있다 → 버그헌터가 이 갭을 채운다

## 7. MVP 범위 (첫 버전에 포함할 것 / 뺄 것)

### ✅ MVP에 포함
- 강사 회원가입/로그인 + 반 생성 + 초대코드
- 학생 회원가입/로그인 + 반 입장
- 강사: 주제 입력 → AI 버그 문제 자동 생성 (3단계)
- 학생: 웹 코드 에디터에서 버그 코드 확인 + 수정
- 학생: 코드 실행 (Pyodide 브라우저 내 Python 실행)
- AI 단계별 힌트 시스템 (3단계 + 최종 해설)
- 강사 대시보드 (학생별 현황, 버그 유형별 성공률)
- 학생 성장 리포트 (기본)

### ❌ MVP에서 제외 (추후 확장)
- Python 외 다른 언어 지원
- 학생간 경쟁/랭킹 시스템
- 실시간 협업 디버깅
- 강사가 직접 버그 문제를 수동 제작하는 기능
- 모바일 반응형 (데스크톱 우선)
- 결제/구독 시스템

## 8. 기술 스택

| 영역 | 기술 | 이유 |
|------|------|------|
| 프론트+백 | Next.js | 풀스택 한방, Vercel 배포 최적화 |
| DB + Auth | Supabase | PostgreSQL + 인증 무료, 빠른 세팅 |
| AI | Claude API (또는 OpenAI GPT-4o) | 코드 분석/생성/힌트에 강함 |
| 코드 에디터 | Monaco Editor | VS Code 기반, 문법 하이라이팅 |
| 코드 실행 | Pyodide (브라우저 내) | 서버 불필요, 무료, 보안 안전 |
| 배포 | Vercel | 무료, Next.js 최적화, 라이브 URL 즉시 발급 |
