## 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | 버그헌터 (BugHunter) |
| 한 줄 소개 | AI가 일부러 버그를 심고, 학생이 찾아 고치는 디버깅 교육 플랫폼 |
| 풀려는 문제 | 모든 교육은 코드 "짜는 법"을 가르치지만, 실무의 60%는 코드 "고치는 법"이다. 아무도 이걸 안 가르친다. |
| 타겟 (메인) | 국비지원/부트캠프 코딩 강사 |
| 타겟 (서브) | 수강생 |

## 핵심 메시지
> "AI가 코드를 짜주는 시대, 개발자의 핵심 역량은 디버깅입니다."

## 기존 서비스와 차별점

| | 프로그래머스/백준 | ChatGPT | 버그헌터 |
|---|---|---|---|
| 목표 | 정답 코드를 처음부터 짜기 | 코드 질문/답변 | 버그 있는 코드를 읽고 고치기 |
| 스킬 | 알고리즘 풀이 | 범용 | 디버깅 사고력 |
| 피드백 | 맞았습니다/틀렸습니다 | 답을 바로 줌 | AI가 사고 과정을 유도하는 힌트 |
| 학습 방식 | 혼자 풀기 | 대화형 | AI 코칭 (답을 안 주고 유도) |
| 실무 연결 | 코딩테스트 대비 | 없음 | 실무 디버깅 역량 훈련 |

## 핵심 기능 (3개)

### 1. AI 버그 코드 생성
- 강사가 주제 입력 (예: "반복문")
- AI가 수준별(초급/중급/고급) 디버깅 문제 자동 생성
- 버그 유형: 문법 에러, off-by-one, 로직 에러, 타입 에러 등

### 2. 3단계 AI 힌트 시스템
- **1단계 (방향 제시)**: "반복문이 몇 번 돌아가는지 세어봐"
- **2단계 (범위 좁히기)**: "5번째 줄의 range 조건을 확인해봐"
- **3단계 (거의 답)**: "range(1,10)은 9까지야. 10을 포함하려면?"
- **해설 모드**: 정답 + 버그 유형 설명

### 3. 강사 대시보드
- 반 전체 실시간 풀이 현황
- 학생별 힌트 사용 횟수 기록
- 가장 많이 틀린 버그 패턴 분석
- "고전 중인 학생" 자동 하이라이트

## 유저 플로우

### 강사 플로우
1. 가입/로그인
2. "반 만들기" → 학생 초대 링크 생성
3. "새 세션 만들기" → 주제 입력 → AI가 수준별 문제 자동 생성
4. 미리보기 → 배포
5. 실시간 대시보드로 현황 확인
6. 수업 후 결과 리포트 확인

### 학생 플로우
1. 초대 링크로 가입
2. 최초 진단 퀴즈 5문제 → 레벨 배정
3. 내 레벨에 맞는 디버깅 문제 확인
4. 코드 에디터에서 버그 찾기 → 수정 → 실행
5. 막히면 AI 힌트 요청 (최대 3단계)
6. 통과 시 다음 문제 or 레벨업

### 데모 시나리오 (심사용, 3분)
1. 강사로 로그인 → "반복문" 주제 → AI가 문제 자동 생성 (10초)
2. 초급 학생으로 전환 → 버그 코드 확인 → 실행 → 에러 발생
3. 힌트 요청 → AI 힌트 → 수정 → 통과!
4. 고급 학생으로 전환 → 더 어려운 버그 → 다른 수준의 힌트
5. 강사 대시보드 → 실시간 현황 + 결과 분석

## 기술 스택

| 영역 | 선택 | 이유 |
|------|------|------|
| 프론트엔드 | Next.js + Monaco Editor | 풀스택 한 프로젝트, 코드 에디터 내장 |
| 백엔드 | Next.js API Routes | 별도 서버 불필요 |
| DB | Supabase (PostgreSQL) | 무료 티어, 인증 내장 |
| AI | OpenAI GPT-4o | 코드 분석/생성 최적 |
| 코드 실행 | Pyodide (브라우저 내 Python) | 서버 불필요, 무료, 보안 안전 |
| 배포 | Vercel | Next.js 최적, 무료, 즉시 라이브 URL |

## MVP 범위

### 포함 (Must Have)
- Python 디버깅 문제 생성
- 3단계 힌트 시스템
- 웹 코드 에디터 + 브라우저 내 실행
- 강사 대시보드 (기본)
- 학생 레벨 시스템

### 제외 (Not in MVP)
- 다중 프로그래밍 언어 지원
- 모바일 반응형
- 이메일 인증
- 학생 간 랭킹/비교
- 복잡한 결제 시스템
